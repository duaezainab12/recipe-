# ⚖️ Lawyer Portal System

A professional **Lawyer Case Management System** designed to streamline legal workflows, manage cases efficiently, and provide a clean dashboard for both lawyers and clients.

---

## 🚀 Features

* 🔐 Secure Login & Registration System
* 👨‍⚖️ Lawyer Dashboard (Admin Panel)
* 👥 Client Management
* 📂 Case Management System
* 📎 File Attachments per Case
* 📅 Court Date Scheduling
* 📊 Case Status Tracking (Pending / Ongoing / Closed)
* 🔍 Smart Search & Filter Cases
* 📝 Editable Cases (Admin Only)
* 👁️ Read-Only Access for Clients
* 📄 Court Order PDF Generation
* 🌙 Dark Mode Toggle
* 🔑 Change Password Option

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP (Laravel)
* **Database:** MySQL
* **Tools:** Git, GitHub

---

## 📁 Project Structure

* `app/` → Core application logic
* `routes/` → Application routes
* `resources/views/` → UI templates
* `public/` → Public assets
* `database/` → Database migrations

## ⚙️ Installation Guide

1. Clone the repository:

```
git clone https://github.com/your-username/recipe.git
```

2. Navigate to project folder:

```
cd Web_FinalProject
```

3. Install dependencies:

```
composer install
```

4. Setup environment:

```
cp .env.example .env
php artisan key:generate
```

5. Run the project:

```
php artisan serve
```
